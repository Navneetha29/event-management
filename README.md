# Event Management System

## Overview
The Event Management System is a web-based application developed as part of my internship practice and learning experience. The purpose of this project was to understand the concepts of full-stack web development and gain hands-on experience in building a complete application. This project was created for educational purposes and includes dummy/sample data for testing and demonstration.

The system helps users manage event-related activities such as creating events, viewing event details, participant registration, scheduling, and managing event information through a centralized platform.

## Purpose of the Project
This project was developed during internship practice to:

- Understand full-stack web development workflow
- Gain practical exposure to application development
- Learn frontend and backend integration
- Practice database management concepts
- Improve problem-solving and coding skills
- Understand project structure and implementation
- Work with APIs and user interactions

This project is built mainly for learning purposes and not intended for real-world production usage.

## Features

- User Registration and Login
- Event Creation and Management
- View Available Events
- Event Scheduling
- Participant Registration
- Venue Information Management
- Event Details Dashboard
- Dummy Event Data Integration
- Responsive User Interface
- Database Connectivity
- Basic CRUD Operations
